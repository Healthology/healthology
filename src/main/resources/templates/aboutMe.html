About Us

We are a group of dedicated Software Developers and this is our final project at Codeup. Healthology is a full-stack Java application that uses Spring to access a MySQL database. The front-end layout is a mixture of BootStrap framework and a lot of custom written CSS. This project utilized two separate APIs- both TalkJS and FileStack were implemented.
